Golang Web Vulnerability Playground

  • Path Traversal
  • RCE
  • SQL Injection
  • XSS
  • Redirect
  • Server-side Template Injection

This is a web vulnerability playground written in Golang. Each link corresponds to a specific vulnerability. Clicking on the link will trigger the vulnerability and show the results. The payload that triggers each vulnerability is provided in the comments of the Golang code.
